# AI 代码规则 (AI Code Rules)

- 性能预算：所有 AI 处理必须在每帧 2ms 内完成
- 数据驱动参数：AI 行为参数（检测范围、反应时间、巡逻速度）必须从外部配置文件加载
- 可调试性：所有 AI 必须暴露调试可视化选项 — 当前状态、检测半径、目标选择
- 预警机制：敌人攻击前必须提供至少 0.5 秒的预警/起手动画（玩家可见的意图提示）
- 分层行为：复杂 AI 行为使用分层状态机或行为树，不允许深度嵌套的 if-else
- 共享查询：多个 NPC 应共享寻路查询和视锥检测以保持在性能预算内
- 回退行为：AI 必须在路径失败、目标丢失或卡住时有优雅的回退行为
