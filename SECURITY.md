# 安全说明

## 关于本项目的安全性声明

**本 Skill 是一个 AI 提示词集合项目**，不包含任何可执行代码、运行时库或后台服务。
它由 Markdown 文档和少量 PowerShell 脚本（钩子脚本）组成，全部内容都可被用户直接
阅读和审计。

**本项目目前没有实施任何自动化的安全检查或安全审计流程。** 所有内容均为"所见即所得"
—— 你可以直接打开文件查看全部内容。

## ⚠️ 使用前的重要提醒

### 1. AI 生成代码的风险

本 Skill 会通过 AI 为你生成游戏代码。请注意：

- **AI 生成的代码可能存在安全漏洞**，包括但不限于：SQL 注入、缓冲区溢出、输入验证缺失、不安全的反序列化等
- **不要在生产环境中直接部署 AI 生成的代码**而不经过人工审查
- **始终对你项目中的代码安全负责**，AI 仅提供建议

### 2. PowerShell 钩子脚本

`hooks/` 目录下的 `.ps1` 脚本由本 Skill 从原版 Bash 脚本改造而来：

- 所有脚本代码**公开透明**，建议在执行前先阅读脚本内容
- 这些脚本**不会自动运行**，需要你手动在 PowerShell 中执行
- 执行前请确认你理解脚本在做什么
- **不要以管理员身份运行这些脚本**，除非你确定需要且理解其操作

### 3. 不要共享敏感信息

在与 AI 协作过程中：

- **不要**在对话中粘贴真实 API 密钥、数据库密码、令牌等机密信息
- **不要**将包含敏感信息的 `.env` 文件内容直接贴入对话
- AI 对话内容可能被用于模型训练或日志记录
- 使用环境变量或独立的配置文件管理敏感信息
