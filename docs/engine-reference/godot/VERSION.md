## 知识缺口警告

LLM 的训练数据可能仅覆盖到 Godot ~4.3 版本。4.4、4.5
和 4.6 版本引入了重大变更，模型并不了解这些内容。
在建议使用 Godot API 调用时，务必交叉参考本目录。

## 截止后版本时间线

| 版本 | 发布 | 风险等级 | 主要主题 |
|------|------|----------|----------|
| 4.4 | ~2025 年中 | 中等 | Jolt 物理选项、FileAccess 返回类型、着色器纹理类型变更 |
| 4.5 | ~2025 年末 | 高 | 无障碍访问（AccessKit）、可变参数、@abstract、着色器烘焙器、SMAA |
| 4.6 | 2026 年 1 月 | 高 | Jolt 默认物理、辉光重做、Windows 默认 D3D12、IK 恢复 |
