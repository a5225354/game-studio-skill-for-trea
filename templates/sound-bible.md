## 混音

### 混音总线结构（Mix Bus Structure）
| 总线 | 内容 | 目标电平 |
| ---- | ---- | ---- |
| Master | 所有内容 | 0 dB |
| Music | 所有音乐轨道 | [目标 dBFS] |
| SFX | 所有音效 | [目标 dBFS] |
| Dialogue | 所有语音/旁白 | [目标 dBFS] |
| UI | 所有界面声音 | [目标 dBFS] |
| Ambient | 环境循环 | [目标 dBFS] |

### 混音规则
- 对白始终优先——对白播放期间压低音乐和音效
- 音乐应当被感受到，而非占据主导——如果玩家无法在音乐声中听到音效，说明音乐太大声
- Master 输出绝对不能削波（Clip）——在 Master 总线上使用限幅器（Limiter）
- 所有音量必须允许玩家调节（按总线）
- 默认混音在扬声器和耳机上都应听起来良好

## 无障碍性（Accessibility）

- 所有关键音频提示必须有视觉替代方案（字幕、屏幕闪烁、图标）
- 为听力障碍玩家提供单声道音频选项
- 所有总线的独立音量控制
- 禁用突发巨大声音的选项
- 所有对白支持字幕，并标注说话者身份
